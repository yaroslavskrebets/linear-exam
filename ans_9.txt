СВОБОДНЫЙ ВЕКТОР — вектор, начало которого может быть совмещено с любой точкой пространства, в котором рассматривается данный вектор. СВОБОДНЫЙ ВЕКТОР можно переносить параллельно самому себе в любую точку пространства. Таким образом, СВОБОДНЫЙ ВЕКТОР задается своей длиной и направлением. Примерами СВОБОДНЫЙ ВЕКТОР могут служить скорость и ускорение твердого тела, движущегося поступательно.

Сложение векторов (сумма векторов) a + b есть операция вычисления вектора c, все элементы которого равны попарной сумме соответствующих элементов векторов a и b, то есть каждый элемент вектора c равен:
сi = ai + bi

Вычитание векторов (разность векторов) a - b есть операция вычисления вектора c, все элементы которого равны попарной разности соответствующих элементов векторов a и b, то есть каждый элемент вектора c равен:
сi = ai - bi

Умножение числа на вектор:
c = alpha * a
ci = alpha * ai
